# Station Management System

## باللغة العربية

هذا نظام لإدارة المحطات. يتضمن هذا النظام عدة ميزات تسمح لك بإدارة المحطات بكفاءة.

## In English

This is a Station Management System. This system includes several features that allow you to manage stations efficiently.

---

### Features:
- إدارة المحطات
- إدارة المستخدمين
- تقارير مفصلة

### Installation:
1. قم بتحميل النظام.
2. اتبع التعليمات لتثبيت البرنامج.

### Usage:
- سجّل الدخول باستخدام بيانات اعتمادك.
- استخدم الواجهة لإدارة المحطات.
